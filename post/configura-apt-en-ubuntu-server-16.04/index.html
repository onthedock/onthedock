Configura el proxy para APT en Ubuntu Server 16.04

  • 10 Jan 2017
  • post
  • Xavier Aznar

Cómo configurar apt para salir a internet a través de un proxy que requiere autenticación.

La configuración del proxy para APT en Ubuntu Server 16.04 se realiza a través del fichero /etc/apt/apt.conf.

Crea el fichero si no existe y escribe:

Acquire::http::Proxy "http://${USERNAME}:${PASSWORD}@proxy.ameisin.vwg:8080/amisin.pac";

A continuación, ya puedes actualizar los repositorios usando apt-get update.
